# Kebijakan Privasi PASSME

**Terakhir Diperbarui: 22 Desember 2024**

PASSME ("kami", "aplikasi") berkomitmen untuk melindungi privasi dan keamanan data Anda. Aplikasi ini dirancang dengan prinsip **Keamanan Lokal Utama**, yang berarti data Anda adalah milik Anda sepenuhnya.

## 1. Informasi yang Kami Kumpulkan
PASSME adalah aplikasi pengelola kata sandi offline. 
- **Data Kata Sandi**: Semua nama pengguna, kata sandi, dan catatan yang Anda simpan dienkripsi menggunakan enkripsi AES-256 dan disimpan secara lokal di perangkat Anda.
- **Data Biometrik**: Aplikasi menggunakan fitur biometrik perangkat Anda (Sidik Jari/FaceID) untuk otentikasi. Kami **tidak memiliki akses** ke data biometrik Anda; proses otentikasi ditangani sepenuhnya oleh sistem operasi perangkat.

## 2. Penyimpanan Data
Semua data didekripsi hanya di dalam memori perangkat saat Anda membukanya. Kami tidak memiliki server pusat, dan kami **tidak pernah mengirimkan** data kata sandi Anda ke server manapun atau pihak ketiga mana pun.

## 3. Keamanan Data
Data Anda dilindungi oleh:
- Enkripsi tingkat militer (AES).
- Kunci enkripsi unik yang dihasilkan pada perangkat Anda.
- Proteksi tambahan melalui autentikasi biometrik.

## 4. Ekspor Data
Fitur ekspor data memungkinkan Anda untuk mencadangkan data ke format CSV. File ini tidak terenkripsi setelah diekspor, jadi Anda bertanggung jawab sepenuhnya atas keamanan file hasil ekspor tersebut.

## 5. Perubahan Kebijakan
Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan berlaku segera setelah dipublikasikan di dalam aplikasi atau melalui toko aplikasi.

## 6. Hubungi Kami
Jika Anda memiliki pertanyaan tentang Kebijakan Privasi ini, silakan hubungi pengembang melalui kontak yang tertera di Google Play Store.
